# Personal Portfolio Website – Arthi P

This is a **simple personal portfolio website** created using **HTML and CSS**.
It showcases my academic details, technical skills, certifications, and achievements as a **2nd year B.E. Computer Science and Engineering student** at Bannari Amman Institute of Technology.

---

## Features

* Clean and professional design
* Fully responsive layout
* Separate HTML and CSS files
* Easy to edit and customize
* Beginner-friendly structure

---

## Technologies Used

* **HTML5** – Structure of the webpage
* **CSS3** – Styling and layout

---

## Project Structure

```
portfolio/
│
├── index.html     # Main HTML file
├── style.css      # External CSS file
└── README.md      # Project documentation
```
